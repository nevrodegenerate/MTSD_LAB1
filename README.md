# Опис застосунка
Консольний застосунок, що приймає шлях до текстового файлу та перетворює його Markdown розмітку в html. 

# Як запустити
На ОС Windows з підтримкою C++, відкрити MTSD_LAB1\x64\Debug\MTSD_LAB1.exe в bash або командній строці Windows, вказати шлях до вхідного файлу та опціональні аргументи.

# Інструкція використання
При запуску необхідно вказати принаймні один аргумент (шлях до вхідного файлу), після чого програма виведе вихідний текст в консолі. 
Якщо після першого аргумента, вказано аргумент --out та шлях та назву вихідного файлу, натомість текст не виведеться та буде створено новий файл.
